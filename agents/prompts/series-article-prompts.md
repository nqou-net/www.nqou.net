# /series-article ワークフロー用プロンプトテンプレート

> 対象ワークフロー: `.agent/workflows/series-article.md`

---

## 基本形（最短）

```
@[/series-article] 
```

> 構造案ファイルを開いた状態で実行すると自動検出されます

---

## 形式指定付き

```
@[/series-article] 統合版でお願いします
```

```
@[/series-article] 連載版（個別記事）でお願いします
```

---

## 構造案と公開日時を指定

```
@[/series-article]

構造案: xxx-series-structure.md
形式: 統合版
公開日時: 2026-02-10T10:00:00+09:00
```

---

## 複数オプション指定

```
@[/series-article]

- 構造案: xxx-series-structure.md
- 形式: 統合版
- 採用案: 案A
- 公開日時: 明日10時
- 挿絵: あり
```

---

## パラメータ一覧

| 項目 | 必須 | デフォルト | 説明 |
|------|------|-----------|------|
| 構造案ファイル | ○ | アクティブタブから自動検出 | `agents/structure/*.md` |
| 形式 | ○ | 確認ダイアログで選択 | 統合版 / 連載版 |
| 採用案 | △ | 構造案内の推薦案 | 案A / 案B / 案C など |
| 公開日時 | ○ | 確認ダイアログで選択 | ISO 8601形式推奨 |
| 挿絵 | - | なし（後から追加可） | `/series-unified-visual` で追加 |

---

## Tips

1. **構造案ファイルを開いてから実行** — 自動検出で入力を省略できる
2. **形式を最初に伝える** — 対話のターン数を削減
3. **公開日時はISO形式推奨** — `2026-02-01T10:00:00+09:00`
4. **挿絵は後から追加可能** — 時間がない時はスキップしてOK
