## 調査結果まとめ

### 共通する重要要素
1. **ガイドラインはプロンプト最適化の対象**: 決まった仕様ではなく、プロンプトエンジニアリングのベストプラクティスを適用する
2. **AIに書かせて観察するアプローチ**: 「わからない」を解決する実践的な方法
3. **うまく動く確率が高い要素**:
   - 名前の一致
   - 定義場所の明示
   - ワークフローの定義
   - 正しい情報提供
4. **実例の重要性**: AGENTS.mdとカスタムエージェントの具体例を示す
5. **現状の課題**: エージェント配置の標準化問題

- **案B**: 失敗と成功のパターン分析。体系的な理解を促す
