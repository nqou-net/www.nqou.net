# agents/templates — 記事作成テンプレート集

このディレクトリには、記事作成時に使用するテンプレートが格納されています。

## テンプレート一覧

| ファイル名 | 用途 | 説明 |
|-----------|------|------|
| [post.md](post.md) | 投稿テンプレート | 単体記事・シリーズ記事の作成用 |
| [warehouse.md](warehouse.md) | 調査ドキュメント | 調査・情報収集の結果をまとめる用 |
| [structure.md](structure.md) | 連載構造案 | シリーズ記事の構造を設計する用 |

## 使用方法

1. 目的に応じたテンプレートをコピーする
2. `<placeholder>` 部分を実際の内容に置き換える
3. WORKFLOWS.md の品質基準に従ってレビューする

### 調査ドキュメント分析（content/warehouse/）

共通構成:
- **要点**: 調査結果の簡潔なまとめ
- **根拠**: 要点を支持する事実やデータ
