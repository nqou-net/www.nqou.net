## 必要な依存関係

- Perl 5.36以上
- Moo（オブジェクト指向フレームワーク）
- Test::More（テストフレームワーク）
- Getopt::Long（コマンドライン引数処理）

Debianベースのシステムでは以下でインストール可能：
```bash
sudo apt-get install libmoo-perl
```

## テスト実行方法

### 全テストを実行
```bash
for dir in 01 02 03 04; do
    echo "=== Testing episode $dir ==="
    cd $dir && prove -lv t/ && cd ..
done
```

### 個別にテスト
```bash
cd 01  # 任意のディレクトリ
prove -lv t/
```

## スクリプト実行例

### 第1回: 基本的なブルートフォース
```bash
cd 01
perl cracker_v1.pl
```

### 第2回: 再帰を使った実装
```bash
cd 02
perl cracker_recursive.pl
```

### 第3回: Iteratorパターン
```bash
cd 03
perl cracker_iterator.pl
```

### 第4回: ポリモーフィズム
```bash
cd 04
# ブルートフォースモード
perl cracker_poly.pl --mode=brute

# 辞書攻撃モード
perl cracker_poly.pl --mode=dict
```

## 学習の進め方

1. **第1回から順番に**読むことをお勧めします
