# 第2回: 再帰を使ったブルートフォース攻撃

## 概要

再帰関数を使って、任意の長さのパスワードに対応できるブルートフォース攻撃の実装です。

## ファイル構成

- `lib/PasswordLock.pm` - 3桁のダイヤル錠を模したクラス（第1回と同じ）
- `cracker_recursive.pl` - 再帰を使って任意の長さに対応するスクリプト
- `t/` - テストコード

## 実行方法

```bash
perl cracker_recursive.pl
```

## テスト実行

```bash
prove -lv t/
```

## 学習ポイント

- 再帰関数の実装
- 可変長パスワードへの対応
- forループとの比較（柔軟性 vs. 複雑性のトレードオフ）

## 注意

デフォルトの `$target_length = 4` で実行すると、最大10,000回の試行が発生します。
デフォルトのパスワードが '777' (3桁) のため、4桁探索では見つかりません。
