## Perlの特徴的な機能

- `use experimental qw(signatures)` - サブルーチンシグネチャ
- `Moo` - 軽量なオブジェクト指向フレームワーク
- `lazy` 属性ビルダー - 遅延評価
- `BUILD` / `DEMOLISH` - ライフサイクルフック
