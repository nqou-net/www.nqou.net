# 第1回：ダンジョン探索ログを作ろう

## 概要
ローグライクゲームで敵を倒したらログに記録する簡単な仕組みを作成。Perlのsayで直接出力する素朴な実装。

## 検証結果

### ✅ 構文チェック
```
game.pl syntax OK
```

### ✅ 実行結果
```
=== ダンジョン探索開始 ===

[LOG] 勇者: スライムを倒した！
[LOG] 勇者: 薬草を手に入れた！
[LOG] 勇者: ゴブリンを倒した！
[LOG] 勇者: レベルが2になった！
[LOG] 勇者: オークを倒した！

=== ダンジョン探索終了 ===
```

## 使用されているクラス
- `GameEvent`: イベントを表現するクラス
- `Game`: ゲームのメインロジック

## 主なメソッド
- `defeat_enemy($enemy_name)`: 敵を倒したときの処理
- `acquire_item($item_name)`: アイテムを取得したときの処理
- `level_up($new_level)`: レベルアップしたときの処理

## 検証者のコメント
このシンプルな実装から、シリーズを通して段階的に洗練されたObserverパターンへと進化していきます。各メソッド内で直接ログ出力を行う素朴な設計は、次回以降で改善されていきます。
