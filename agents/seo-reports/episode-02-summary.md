## 🎓 改善提案

### 構造化データの実装方法

Hugoテンプレートに以下を追加:

```html
<!-- layouts/partials/structured-data.html -->
{{ if in .Params.tags "tutorial" }}
  <!-- HowTo スキーマ -->
  <script type="application/ld+json">
  {
    "@context": "https://schema.org",
    "@type": "HowTo",
    "name": {{ .Title }},
    "totalTime": "PT10M",
    ...
  }
  </script>
{{ end }}

{{ if .Params.faq }}
  <!-- FAQPage スキーマ -->
  <script type="application/ld+json">
  {
    "@context": "https://schema.org",
    "@type": "FAQPage",
    ...
  }
  </script>
{{ end }}
```
