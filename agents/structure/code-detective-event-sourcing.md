- **呼称と質問の型**: 地の文では「男」→「ロック」。パフォーマンスやコストの観点から具体的に反論する

- Repository回（中原・僕）: 自責的で内省的

- **今回（日下部・俺）**: 実務優先で反論的。「それで何ミリ秒遅くなるんですか」型の質問を複数回投げる

### 2. 導入の差分
- 直近3本: LCI事務所訪問（Visitor）/ 会議室に呼ぶ（Repository, Null Object）
- **今回**: 同僚の紹介で、ロックがEC事業部のオフィスに出張してくる。語り部のデスクの前に立つところから始まる

### 3. 対話の差分
- 直近: 困惑→質問→納得のサイクルが比較的スムーズ
- **今回**: 語り部が「毎回全イベント舐めるのか。10万件のイベントがあったら？」とパフォーマンスで反撃。ロックがSnapshot概念を会話の中で自然に返す

### 4. 終わり方の差分
- 直近: 報酬ギャグ（Visitor: コーヒーの代金、Repository: テスト実行回数、Null Object: 会議室の利用時間分）
- **今回**: 報酬ギャグなし。語り部がエレベーターを待ちながら「3年間、俺たちは証拠を消し続けていた側だった」と気づく独白。技術的余韻

### 5. 関係性の差分
- **今回**: 同僚・吉岡の紹介で初対面。吉岡が隣の席から「変な奴だけど腕は確かだ」と耳打ちするのを冒頭で描写

### 6. 質問の差分
- 語り部が理解のための質問を入れるポイント:
  - 「`stock`が属性じゃなくてメソッドになった」→ なぜ毎回計算する必要があるのか
  - 「毎回全イベントをループは遅くないか」→ パフォーマンスへの懸念（ここでSnapshot が返答として登場）
  - 「イベントが不変って、間違って記録したらどうする」→ 補正イベントの概念に触れる

## 5幕構成プロット

### I. 導入（依頼）

**見出し**: 三十日間の空白

- 舞台: EC事業部のオフィス（開放型デスク配置）
- 監査部門からの要求メール「過去30日の在庫変動根拠を提出せよ」がモニターに表示されている
- 同僚の吉岡が「前にうちのAPIのバグを見てもらった奴がいる。変わってるけど腕は確かだ」と紹介
- ロックが事業部に現れる。エナジードリンク片手、PCバッグを肩にかけて。挨拶もなくモニターを覗き込み「三十日分か。記録はあるのか」
- 日下部:「在庫数ならDBにある」ロック:「在庫数ではない。出来事だ」
- 名前を聞く前に「ワトソン君」と呼ばれる。日下部は無視する

### II. 現場検証（Beforeコード）

**見出し**: 上書きされた犯行現場

- ロックが日下部のPCで在庫管理コードを開く（許可は取らない）
- `Item`クラス: `stock`が`rw`で、`add_stock`/`reduce_stock`が直接上書き
- ロック:「完璧な証拠隠滅だ。入荷では`stock`を書き換え、出荷でも書き換え。現場には今の数字だけが残る」
- 日下部:「証拠隠滅って——普通の在庫管理だろう」
- ロック:「その通り。だが普通の在庫管理では、監査には答えられない」
- ホワイトボード（or 紙ナプキン的なもの）にt=0〜t=4の状態遷移を手書きで描く
- 日下部が反論:「でも現在地は正確ですよ」→ ロック:「裁判に証拠なしで臨むつもりかね」

### III. 推理披露（Event Sourcing 適用）

**見出し**: 出来事を積み重ねる捜査手法

- ロック:「優秀な探偵は現場を変えない。出来事を記録するだけだ」
- `StockEvent`クラスの導入。`is => 'ro'`（不変）であることを強調
- `Item`の`stock`を属性からメソッドに変更。イベントのリプレイで計算
- **日下部の反論1**:「毎回全イベントをループするのか。商品1万点×年間イベント数で考えたら——」
  → ロック: Snapshotの概念を会話の中で説明（ただし今回のスコープ外として深追いしない）
- **日下部の反論2**:「イベントを間違って記録したら？ `ro`だから消せないんだろう」
  → ロック:「消すのではなく、補正イベントを追加する。帳簿の赤伝票と同じだ」（概念の紹介のみ）
- `stock($upto)` によるタイムトラベル機能を披露
- 日下部:「……30日前の在庫が、タイムスタンプ一つで出せる」

### IV. 解決（テスト通過）

**見出し**: 時を遡る証明

- テスト結果の表示。全パス、警告ゼロ
- 日下部:「監査部門に30日分出せる」
- Mermaid図でクラス構造を示す
- ロックは荷物をまとめ始める。日下部が「報酬は？」と聞く前に、ロックが「吉岡に借りがあったから、今回は貸しだ」と言い残して去る
- 日下部がエレベーターホールで一人になり、「3年間、俺たちは知らないうちに証拠を消し続けていた」という独白

### V. 報告書（探偵の調査報告書）

- 事件対応表
- 推理のステップ
- ロックより: 標語型でない、より散文的なメッセージ。証拠と出来事の本質に触れる
