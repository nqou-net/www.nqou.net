# Medical Chart
- **Symptom**:
    - Technical: 既存のクライアントコード (`Client`) が期待するインターフェース（単純な文字列引数）と、新しいライブラリ (`ModernLogger`) のインターフェース（ハッシュリファレンス等の構造化データ）が不一致。
    - Metaphor: 「臓器移植（新しいロガーの導入）を行いたいが、血管の結合部（メソッドシグネチャ）の形状が合わず、そのままでは接続できない状態」
- **Diagnosis**: **インターフェース不適合症候群 (Interface Incompatibility Syndrome)**
    - 病状: 無理に接続しようとすれば全身（システム全体）に出血（エラー）が広がり、修正範囲が爆発的に増加する。
- **Prescription**: **Adapter Pattern (変換アダプター処方)**
    - 治療法: 既存の血管と新しい臓器の間に、形状を変換するコネクタ（Adapter）を介在させる。これにより、既存の肉体（クライアントコード）を切り刻むことなく、新しい臓器を機能させる。
