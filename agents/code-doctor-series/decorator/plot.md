# Plot Outline

## Scene 1: The Emergency
深夜のサーバーサイド開発室。壁一面のホワイトボードには、蜘蛛の巣のような巨大なクラス継承図が描かれている。
継承 累 (Keisho Rui) は、ディスプレイの前で頭を抱えていた。「『伝説の剣』に『炎属性』をつけると『LegendaryFireSword』...さらに『耐久度強化』がつくと『DurableLegendaryFireSword』...もうクラス名を考えるだけで日が暮れる！」
その時、サーバーラックの隙間から、黒衣のドクター（Code Doctor）と白衣の助手（Nurse）が音もなく現れる。
ドクター：「ここか。指数関数的な『悲鳴』が聞こえたのは」
累：「だ、誰ですか！？ セキュリティアラートは...」
助手：「アラートが鳴る前に、システム自体が重すぎて沈黙していますね」

## Scene 2: The Misunderstanding (Diagnosis)
ドクターはホワイトボードの継承図を冷ややかな目で見上げる。
ドクター：「美しい家系図だ。近親婚を繰り返しすぎて、血が腐りかけているがな」
助手：「（通訳）継承の深さが限界を超え、少しの変更で全体が壊死する状態です」
ドクターは累の手からマウスを強引に奪い取る。その際、冷たい指先が累の手に触れる。
ドクター：「貸せ。これ以上、その醜悪な『家系』を増やすな」
累：「（なんて冷たい手...でも、僕の迷いを断ち切ってくれる力強さを感じる...！ この人は、僕を救いに来てくれた救世主（メシア）なんだ！）」
累はドクターを熱っぽい目で見つめ始めるが、ドクターは単にキーボードが手垢で汚れているのを嫌がっているだけである。

## Scene 3: The Surgery
ドクターは外科医のように手袋をはめ、リファクタリングを開始する。
ドクター：「メス。Decoratorパターン」
助手：「はい、委譲による動的機能拡張ですね」

**Bad Code (Before):**
```perl
class Weapon { ... }
class Sword extends Weapon { ... }
class FireSword extends Sword { ... }
class IceSword extends Sword { ... }
class FireIceSword extends FireSword { ... } # 多重継承の悪夢
```
ドクター：「この硬直した『継承（is-a）』の鎖を断ち切り、柔軟な『集約（has-a）』の層（レイヤー）で包み込む」

**Good Code (After):**
```perl
my $sword = Sword->new;
$sword = FireAttribute->new( component => $sword );
$sword = IceAttribute->new( component => $sword );
```
ドクターがキーを叩くたび、ホワイトボードの線が次々と消え、代わりにシンプルなオブジェクトがマトリョーシカのように重なっていく映像が比喩的に描かれる。
累：「すごい...クラスが消えていく...！ 装備が、まるで『着せ替え人形』のように自由に...！」

## Scene 4: The Prognosis (Twist & Ending)
治療が終わり、システムは劇的にスッキリした。
累：「これでどんな組み合わせも怖くない！ よーし、全属性・全バフ・全特効を乗せた『アルティメット・ゴッド・ウェポン』を作るぞー！」
累は興奮して、100個以上のDecoratorを二重ループで適用し始める。
助手：「あ、患者さん、それは...」
実行した瞬間、サーバーのファンが轟音を上げ、処理がタイムアウトする。
ドクター：「着ぶくれしすぎて窒息しているな。機能（服）を重ねるのと、処理（体重）が増えるのは等価交換だ」
累：「はっ...自由には責任が伴う...そういうことですね！？（キラキラした目）」
ドクター：「（無視して）行くぞ、助手。次の『スパゲッティ』が待っている」
ドクターたちはサーバーラックの彼方へ消える。
累：「ありがとうございました！ 先生！ ...（誰もいない空間に向かって一礼）...ツンデレなんだから」
助手：（遠くから）「ただのコミュ障ですよー」
