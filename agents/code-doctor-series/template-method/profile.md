- Role: 社内ツール開発チームのバックエンド担当（エンジニア歴3年）
- Personality: 真面目で責任感が強いが、「動けばOK」の精神で目の前のタスクを片付けるタイプ。先輩のコードをコピペして少し変えるのが得意技。レビューで「似たコード多くない？」と言われると「仕様が違うんで……」と苦しい言い訳をする。

# Medical Chart
- Symptom: データエクスポート処理で「取得→変換→出力」の流れが3つのサブルーチンにコピペされ、共通処理の修正が入るたびに3箇所を手作業で同期している。先日、CSV側だけ修正を忘れて不整合データが本番に流れた。
- Metaphor: 「臓器クローン症候群――同じ臓器のコピーが体内に3つあり、1つが感染すると他の2つにも手術が必要になる状態。しかも毎回微妙に血管の繋ぎ方が違うため、執刀医が混乱する」
- Diagnosis: 反復性コピペ壊死（Repetitive Copy-Paste Necrosis）
- Cure: Template Method パターンによる「標準手術プロトコル策定」――共通の処理手順（骨格）を1つの基底クラスに集約し、形式ごとの差異だけをサブクラスに委譲する。
