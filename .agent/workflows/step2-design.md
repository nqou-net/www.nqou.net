# Step2 Design Workflow

## When to Use

Step1完了後（01_interview.md が回答付きで作成されている状態）。
または `/step2-design` で明示的に呼び出されたとき。

## Procedure

1. アーティファクトディレクトリから 01_interview.md を読み込み、リクエストと回答を確認する
2. プロンプト設計書（Prompt Spec）を作成する
   - 記事の目的と成功基準
   - 対象読者と前提知識
   - 記事構成（セクション構成）
   - トーンとスタイル
   - 出力フォーマット
3. 品質基準（RULES.md）に照らしてセルフチェックする
4. システムアーティファクトディレクトリに記録する

## Output

- システムアーティファクトディレクトリ（絶対パス）
  - 02_prompt_spec.md: プロンプト設計書
