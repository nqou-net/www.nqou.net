# Step1 Interview Workflow

## When to Use

ユーザーが「〜について記事を書きたい」などの記事執筆リクエストを入力したとき。
または `/step1-interview` で明示的に呼び出されたとき。

## Procedure

1. リクエストを分析し、必要なパラメータを特定する
2. 質問を生成する（RULES.md に従う）
3. ユーザーに質問を提示し、回答を待つ
4. 質問、選択肢、★推奨、回答をシステムアーティファクトディレクトリに記録する

## Output

- システムアーティファクトディレクトリ（絶対パス）
  - 01_interview.md: リクエスト原文、Q1〜Q3（選択肢、★推奨、回答）
