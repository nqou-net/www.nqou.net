# Code Detective Narrative Skill

このスキルは、「コード探偵ロックの事件簿」シリーズにおいて、読者を惹きつける物語を生成するための**発想の種**です。

> **大原則: ここに書かれた内容は「ルール」ではなく「レシピ集」。材料の配分を変えても、材料を足しても引いても、出来上がりが美味しければそれが正解。**

---

## 1. 探偵メタファーの等価変換技術

アンチパターンとデザインパターンを探偵用語に変換するための**起点**。

### アンチパターン → 事件・犯人メタファー（例）

| 技術的特徴 | メタファーの種 |
|---|---|
| God Class（何でも背負い込む） | 暴走する独裁者、全知全能の神が自重で潰れる |
| Spaghetti Code（絡み合って解けない） | 密室殺人、誰も解けない迷宮 |
| Shotgun Surgery（一箇所の変更が全体に散弾銃） | 連続爆破事件、ドミノ崩し |
| Lava Flow（触れない化石コード） | 発掘された遺跡、動かせない呪いのオブジェ |

### デザインパターン → 推理・切り札メタファー（例）

| パターン | メタファーの種 |
|---|---|
| Strategy | 戦略の切り替え、犯人に合わせた捜査変更 |
| Observer | 監視網の構築、情報の即時伝達 |
| Facade | 窓口の一本化、事件の裏の黒幕への直通ルート |
| Chain of Responsibility | リレー捜査、たらい回しからの脱出 |

**自由に拡張すること。** 上記にないメタファーを発明しても構わない。

## 2. 「巻き込まれコメディ」の構築レシピ

ロックとワトソン君の掛け合いの**基本骨格**。ただしこの骨格は折っても曲げてもいい。

### 基本パターン（出発点）

1. **Trigger**: ロックが探偵として当然のように行動する
2. **Bewilderment**: ワトソン君が戸惑う
3. **Escalation**: ロックが気にせず推理を進める
4. **Acceptance**: ワトソン君が流されつつも助手として機能し始める

### 変奏例（むしろこういうのが面白い）

- **逆転パターン**: ワトソン君が先に気づき、ロックが後追いする回
- **対立パターン**: ワトソン君がロックの推理に反論し、別の解決策を提案する回
- **不在パターン**: ロックが不在で、ワトソン君が過去の事件ファイルを読み返す形式
- **複数ワトソン**: 2人の依頼人が同時に来て、ロックが両方を「ワトソン君」と呼ぶ混乱
- **ロック敗北**: ロックの最初の推理が外れ、二度目のアプローチで解決する回

**要点**: ロックは「迷惑なだけの人」にならない。ワトソン君は「何も分からない素人」にならない。この2点だけ守れば、あとは自由。

## 3. シーン構成のペース配分

以下は**デフォルトの配分**。物語の面白さのために大胆に崩してよい。

| Phase | デフォルト | 変形例 |
|---|---|---|
| I. 依頼 | 15% | 冒頭で事件解決済み → 回想形式で0%から始める |
| II. 現場検証 | 25% | 検証が物語の核 → 60%に拡大。推理は短く鮮やかに |
| III. 推理披露 | 40% | 推理を省略し、結果だけ見せる。解説はワトソンの独白で |
| IV. 解決 | 20% | 解決しない。次回に続く。（連載向け）|

### 構成の自由度

- 時系列を入れ替えてもいい（結末→回想→真相）
- 手記・報告書・メール形式で書いてもいい
- ロックの視点を一瞬だけ挟んでもいい（ワトソン君が「彼は何を考えていたのだろう」と推測する形で）

## 4. ワトソン君の困惑設計ガイド

### 核となる性質

**「知識はあるがひらめきがない」** — これが基本。ただし毎回同じパターンにならないよう、バリエーションを持たせる。

### 困惑の表現パターン集

| パターン | 例 |
|---|---|
| **素朴な疑問型** | 「クラスが分かれたのは分かりますが、どう切り替えるんですか？」 |
| **先走り失敗型** | ワトソン君が推測するが的外れ → ロックが正解を示す |
| **遅延理解型** | ロックの説明を聞いた後、2テンポ遅れて「……あっ！」と理解 |
| **部分理解型** | 半分は分かるが残り半分が分からず、追加の質問をする |
| **逆転型（稀に使用）** | ワトソン君が核心に触れ、ロックが「……君、なかなかやるじゃないか」|

### DON'T（これだけは避ける）

- ロックより先に**毎回**解決策に気づく（稀な逆転回は除く）
- 一度の説明で全てを完全に理解する
- 技術的に無知すぎる反応（「クラスって何ですか？」）

## 5. Sub-Agent Roles（執筆時の人格切り替え）

- **The Screenwriter (脚本家)**: プロットの骨組み、伏線、オチ。「この回の面白さの核は何か？」を常に問う。
- **The Translator (翻訳家)**: 技術用語を探偵用語に変換。自然な日本語を選ぶ。
- **The Director (演出家)**: ト書き（描写）の追加。「ここでロックにコートの襟を立てさせる」「ここでワトソン君に溜息をつかせる」。
- **The Rebel (反逆者)**: 「これ、逆にしたら面白くない？」と常識を疑う。配分を崩す、設定を裏切る、読者の予想を外す。

> **The Rebel を忘れないこと。** 安全な定型を繰り返すより、一つの大胆な試みの方が読者の記憶に残る。
