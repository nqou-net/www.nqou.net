# プロット構成: 医療ドラマ（4幕構成）

## 概要
コードドクターシリーズにおける記事全体の流れ（プロット構造）を定義します。
技術的な解説を、患者がクリニックを訪れてから退院するまでの4つのフェーズに沿って展開します。

**見出しルール**: 見出しは**医療メタファーのみ**使用。「第1章」や番号、順序を直接的に数字で示すのは**禁止**。

## 構成案

| フェーズ | 見出しのメタファー例（1〜2個選択） | ナラティブの目標（何を描くか） |
| :--- | :--- | :--- |
| **I. 導入** | 来院, 救急搬送, 主訴, 往診 | コードの問題（症状）を抱えて来院または往診依頼。患者の心理状態（不安/期待）と、ドクターに対する初対面の警戒感を描写する。 |
| **II. 検査** | 触診, 画像診断, 診断 | ドクターが解析（Beforeコードの確認）し、助手が通訳する。アンチパターンを「病名」として特定する。 |
| **III. 処置** | 処方箋, 外科手術, 集中治療 | デザインパターンを適用。リファクタリング（Afterコードの実装＝手術）の実施。**ドクターが黙々と作業し、ナナコが解説**する。 |
| **IV. 予後** | 術後経過, リハビリテーション, 退院指導 | 改善結果（テスト通過・可読性向上）の確認。別れの挨拶とコミットメント（報酬の約束）。 |
