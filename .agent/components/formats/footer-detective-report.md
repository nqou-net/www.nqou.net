# 出力形式: 探偵の調査報告書

## 概要
記事の最後に水平線（`---`）を挟んで「探偵の調査報告書」を含めること。
事件の総括と、読者への具体的な解決策（GoFパターン）の適用方法をまとめます。

## Format
1.  **Separator**: `---`
2.  **Heading**: `## 探偵の調査報告書`
3.  **Table**: 事件と解決策
    *   Columns: `| 容疑（アンチパターン） | 真実（パターン） | 証拠（効果） |`
4.  **List**: 推理のステップ（リファクタリング手順）
    *   Heading: `### 推理のステップ`
    *   読者が自身で適用するためのステップバイステップの解説。
5.  **Message**: ロックからワトソン君へ
    *   Heading: `### ロックより`
    *   Content: 探偵らしいキザな口調による総括と、次なる事件への期待（1〜2段落程度）。
