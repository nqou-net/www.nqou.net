# メタファー: 医療（ドクター）

## 概要
技術的な課題や解決策を、医療、病気、手術などの専門的な医療用語に置き換えて表現します。

## 用語マッピング

| 技術用語（実態） | メタファー（比喩） | 例 |
| :--- | :--- | :--- |
| **アンチパターン（原因）** | 病名、症候群 | 多重人格症候群、巨大腫瘍 |
| **既存のコード（Before）** | 患者の体、患部 | 「結合度が高すぎて、癒着が激しい」 |
| **症状・バグ** | 主訴、痛み、出血 | 「少し動かすだけでデグレ（出血）する」 |
| **コードの解析** | 触診、画像診断、検査 | ドクターがコードをスキャンする |
| **デザインパターン（解決策）** | 処方箋、治療法、特効薬 | Strategyパターンというバイパス手術 |
| **リファクタリング** | 外科手術、集中治療 | ドクターが黙々とキーボードを叩く（メスを入れる） |
| **テストの通過・改善** | 術後経過、リハビリ、退院 | 「メトリクスが正常値に戻った」 |

## タイトル命名規則 (Title Naming Convention)

### フォーマット
```
コードドクター【{PatternName}】{MedicalMetaphor}〜{ThemeMetaphor}〜
```

| 要素 | 説明 | 例 |
|------|------|-----|
| **PatternName** | デザインパターン英語名 | Singleton, Factory Method |
| **MedicalMetaphor** | 症状・病名・処置 | 多重人格症候群、癒着分離手術 |
| **ThemeMetaphor** | 技術課題の比喩 | 唯一の真実、疎結合の誕生 |

## 演出のルール
*   **クリニックの描写**: 雑居ビルの2階、目立たない場所。看板はなく、ドアのプレートに「コード診療所」と書いてあるだけ。内装は元ネイルサロン。ドアを開けると本の山が目前に迫る。
*   **専門用語と通訳**: ドクターは専門的な技術用語（医療用語に見立てたもの）のみを呟き、助手がそれを日常的な比喩（料理、建築など）に翻訳して患者に説明する。
*   **コメディ要素**: ドクターが助手の些細な親切（コーヒー、ペン渡し）を自分への「好意」と勘違いするシーンを1回入れる。ただしドクターの心の声は描写せず、気まずい沈黙や行動で示すこと。
