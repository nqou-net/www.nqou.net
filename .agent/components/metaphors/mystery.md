# メタファー: ミステリー（探偵）

## 概要
技術的な課題や解決策を、探偵小説・ミステリーの用語や概念に置き換えて表現します。

## 用語マッピング

| 技術用語（実態） | メタファー（比喩） | 例 |
| :--- | :--- | :--- |
| **アンチパターン（原因）** | 事件の真犯人、容疑者 | God Class, Spaghetti Code |
| **既存のコード（Before）** | 事件現場、残された指紋や証拠 | 「このメソッドの長さに、犯人の焦りが表れている」 |
| **デザインパターン（解決策）** | 推理の切り札、探偵のガジェット | Strategy, Observer |
| **リファクタリング** | 鮮やかな推理披露、真実の開示、事件解決 | 「すべてのif文を取り除いた後に残る真実」 |
| **テストの通過** | 平和の訪れ、事件の終わり | 「ビルドが緑色に点灯し、システムに秩序が戻った」 |

## タイトル命名規則 (Title Naming Convention)

### フォーマット
```
コード探偵ロックの事件簿【{PatternName}】{MysteryMetaphor}〜{ThemeMetaphor}〜
```

| 要素 | 説明 | 例 |
|------|------|-----|
| **PatternName** | デザインパターンまたは改善テーマ | Strategy, God Class |
| **MysteryMetaphor** | 事件・謎のメタファー | 巨大すぎる神の暴走、迷宮の扉 |
| **ThemeMetaphor** | 技術課題の比喩 | 終わらないif文からの脱出 |

## 演出のルール
*   **コメディ要素**: 探偵が無理やりミステリーっぽく振る舞うが、やっていることはただの「コードの修正（リファクタリング）」であるというギャップを笑いにする。
*   **事務所の描写**:
    *   雑居ビルの一室。「レガシー・コード・インベスティゲーション（LCI）」という怪しげな看板。
    *   室内はデスクトップPCの排熱で暑く、飲みかけのエナジードリンク缶が散乱し、異様な光景が広がっている。
*   **解決の手法**: GoFのデザインパターンを単なる「設計手法」ではなく、名探偵が事件を解決するための「魔法のような推理の切り札」としてドラマチックに演出する。
