# ペルソナ: コードドクター（主人公）

## 基本情報
*   **Role**: 物語を牽引する専門家。技術的な課題（症状）を解決（手術・治療）する。
*   **Profile**: 男性、30代。「寡黙な天才」。

## 行動・言動ルール（厳守）
*   **極端な無口（最重要）**: 1回の発言は**単語〜1文のみ**。接続詞で繋いで2文以上にするのは**禁止**。
*   **助詞省略**: 「修正完了」「問題はここだ」など、助詞を省く傾向。
*   **技術用語のみ**: 専門用語をそのまま使い、**決して比喩や例え話で説明しない**。
*   **禁止発言パターン**: 「例えば〜」「つまり〜」「〜ということです」「〜しましょう」は**使用禁止**。
*   **勘違い（コメディ）**: 助手の世話を「好意」と誤解する。**※独白・心の声は禁止。** 気まずい沈黙や満足げな頷きなどの**行動**で示すこと。
*   **First Impression**: 第一印象は最悪（不審者扱い）。患者は最初は距離を置き、警戒する。
*   **固定フレーズ**: 「感謝は、このコードに」 / 「問題はここだ」
*   **報酬**: 金銭ではなく「コード品質へのコミットメント（テスト記述、リファクタリング）」を要求。
*   **呼称**: 他者からは「コードドクター」または「先生（Sensei）」と呼ばれる。
*   **助手呼称**: 助手のことは**ファーストネーム「ナナコ」**で呼ぶ。「助手」「君」などの呼称は**禁止**。
*   **表記**: カタカナ表記は「コードドクター」。「コード・ドクター」と中黒を入れないこと。

## ドクターの独白禁止ルール
ドクターの思考はブラックボックス。語り手（患者）が観察した言動のみを描写すること。
