## 今回のゴール

今回は、固定データを画面に表示するシンプルなスクリプトを作成します。まずは動くものを作って、レポート生成の基本的な流れを体験しましょう。

## 最小限のレポート生成スクリプト

まずは、もっともシンプルな形でレポートを表示してみましょう。

言語・バージョン: Perl v5.36以降  
外部依存: なし

```perl
#!/usr/bin/env perl
use v5.36;

# 固定データ
my @data = (
    { name => '田中太郎', score => 85 },
    { name => '鈴木花子', score => 92 },
    { name => '佐藤一郎', score => 78 },
);

# レポートを表示
say '=== 成績レポート ===';
say '';

for my $row (@data) {
    say "$row->{name}: $row->{score}点";
}

say '';
say '=== レポート終了 ===';
```

実行方法:
```bash
perl report.pl
```

出力例:
```
=== 成績レポート ===

田中太郎: 85点
鈴木花子: 92点
佐藤一郎: 78点

=== レポート終了 ===
```

このスクリプトは動きますが、いくつかの問題があります。

- データがスクリプト内に直接書かれている（ハードコーディング）
- データの加工処理がない
- 出力形式を変更するのが難しい

これらの問題を解決するために、次回以降でプログラムを改良していきます。
