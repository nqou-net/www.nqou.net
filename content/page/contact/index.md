# お問い合わせ

記事内容の確認、誤記の連絡、仕事の相談、掲載内容に関する問い合わせは、以下の公開窓口から受け付けています。

## 連絡方法

### メール

メール: [nqou.net@gmail.com](mailto:nqou.net@gmail.com)

非公開で連絡したい内容や、仕事の相談、掲載内容の訂正依頼はメールをご利用ください。

### X

運営者アカウント: [@nqounet](https://x.com/nqounet)

短い連絡や、公開でも差し支えない内容であれば X からご連絡ください。

### GitHub

GitHub: [nqounet](https://github.com/nqounet)

記事の誤字、リンク切れ、コード例の修正提案など、公開で問題ない内容は GitHub 上での連絡が扱いやすいです。

## お問い合わせ時のお願い

- 対象記事または対象ページの URL を記載してください
- 指摘内容は、再現手順や根拠が分かる形で記載してください
- 個人情報、機密情報、パスワードなどは送らないでください
- 件名に要件が分かるキーワードを入れてください
- 返信を約束するものではありません

## 受け付けていない内容

- 営業メールの一斉送信
- 誹謗中傷、脅迫、迷惑行為
- サポート対象外サービスに関する問い合わせ

詳細は [About](/page/about/) をご確認ください。個人情報の取り扱いについては [プライバシーポリシー](/page/privacy-policy/) をご確認ください。
