### アプリの基本構造を設計する

**【コード例2：アプリの基本構造（while無限ループ）】**

```perl
#!/usr/bin/env perl
use strict;
use warnings;
use utf8;

# 標準入出力をUTF-8に設定
binmode STDOUT, ':utf8';
binmode STDIN,  ':utf8';

# メイン処理
while (1) {
    # メニューを表示
    print "\n==== TODOリスト ====\n";
    print "1. タスクを追加\n";
    print "2. タスク一覧を表示\n";
    print "3. タスクを完了にする\n";
    print "4. タスクを削除\n";
    print "0. 終了\n";
    print "選択してください: ";

    # ユーザー入力を受け取る
    my $choice = <STDIN>;
    chomp $choice;

    # 選択に応じた処理
    if ($choice eq '1') {
        print "（タスク追加機能は次回実装します）\n";
    }
    elsif ($choice eq '2') {
        print "（タスク一覧機能は次回実装します）\n";
    }
    elsif ($choice eq '3') {
        print "（完了機能は次回実装します）\n";
    }
    elsif ($choice eq '4') {
        print "（削除機能は次回実装します）\n";
    }
    elsif ($choice eq '0') {
        print "終了します。\n";
        last;  # ループを抜ける
    }
    else {
        print "無効な選択です。もう一度選んでください。\n";
    }
}
```

### コード解説：重要なポイント

#### UTF-8対応
```perl
use utf8;                    # ソースコード内の文字列がUTF-8であることを宣言
binmode STDOUT, ':utf8';     # 標準出力をUTF-8に設定
binmode STDIN,  ':utf8';     # 標準入力をUTF-8に設定
```

日本語を扱うために必須の設定です。

#### 無限ループと脱出
```perl
while (1) {
    # ... メニュー表示と処理 ...
    
    if ($choice eq '0') {
        last;  # ループを抜けてプログラム終了
    }
}
```

`while (1)` は「常に真」なので無限ループになります。`last` でループを脱出します。

#### chomp の重要性
```perl
my $choice = <STDIN>;
chomp $choice;
```

`<STDIN>` で読み込んだ文字列には末尾に改行文字（`\n`）が含まれています。`chomp` で改行を取り除かないと、`$choice eq '1'` のような比較が正しく動作しません。

## 完成形：対話ループの実装

**【コード例2：アプリの基本構造（while無限ループ）】**

```perl
#!/usr/bin/env perl
use strict;
use warnings;
use utf8;
use open ':std', ':encoding(UTF-8)';

# メイン処理
main();

sub main {
    while (1) {
        show_menu();
        
        my $choice = <STDIN>;
        chomp $choice;
        
        if ($choice eq '1') {
            add_task();
        }
        elsif ($choice eq '2') {
            list_tasks();
        }
        elsif ($choice eq '3') {
            complete_task();
        }
        elsif ($choice eq '4') {
            delete_task();
        }
        elsif ($choice eq '0') {
            print "TODOアプリを終了します。\n";
            last;
        }
        else {
            print "エラー: 0〜4の数字を入力してください。\n";
        }
    }
}

sub show_menu {
    print "\n";
    print "=" x 30 . "\n";
    print "  TODOリスト管理\n";
    print "=" x 30 . "\n";
    print "1. タスクを追加\n";
    print "2. タスク一覧を表示\n";
    print "3. タスクを完了にする\n";
    print "4. タスクを削除\n";
    print "0. 終了\n";
    print "=" x 30 . "\n";
    print "選択 (0-4): ";
}

# 各機能のサブルーチン（次回以降で実装）
sub add_task      { print "タスク追加機能（第2回で実装）\n"; }
sub list_tasks    { print "一覧表示機能（第4回で実装）\n"; }
sub complete_task { print "完了機能（第8回で実装）\n"; }
sub delete_task   { print "削除機能（第9回で実装）\n"; }
```


これで `binmode` を個別に書く必要がなくなります。

- メイン処理の見通しが良くなる
- 次回以降、少しずつ実装していける

## 実装：最初の一歩を踏み出そう

**【コード例2：アプリの基本構造（while無限ループ）】**

```perl
#!/usr/bin/env perl
use strict;
use warnings;
use utf8;

# 日本語を正しく扱うための設定
binmode STDOUT, ':encoding(UTF-8)';
binmode STDIN,  ':encoding(UTF-8)';

print "🎉 TODOアプリへようこそ！\n";

# メイン処理ループ
while (1) {
    # メニュー表示
    print "\n" . "=" x 40 . "\n";
    print "📋 TODOリスト - あなた専用タスク管理\n";
    print "=" x 40 . "\n";
    print "  [1] タスクを追加する\n";
    print "  [2] タスク一覧を見る\n";
    print "  [3] タスクを完了にする\n";
    print "  [4] タスクを削除する\n";
    print "  [0] 終了する\n";
    print "=" x 40 . "\n";
    print "👉 選択してください (0-4): ";

    # ユーザーの選択を受け取る
    my $choice = <STDIN>;
    chomp $choice;  # 末尾の改行を削除

    # 選択に応じた処理
    if ($choice eq '1') {
        print "💡 タスク追加機能（次回実装予定）\n";
    }
    elsif ($choice eq '2') {
        print "💡 タスク一覧機能（次回実装予定）\n";
    }
    elsif ($choice eq '3') {
        print "💡 完了機能（次回実装予定）\n";
    }
    elsif ($choice eq '4') {
        print "💡 削除機能（次回実装予定）\n";
    }
    elsif ($choice eq '0') {
        print "\n👋 TODOアプリを終了します。お疲れ様でした！\n";
        last;  # ループを抜けてプログラム終了
    }
    else {
        print "❌ エラー: 0〜4の数字を入力してください。\n";
    }
}
```

### コードのポイント

#### 絵文字でUI を親しみやすく
```perl
print "🎉 TODOアプリへようこそ！\n";
```

UTF-8対応していれば絵文字も使えます。視認性が上がります。

#### chomp は必須
```perl
my $choice = <STDIN>;
chomp $choice;
```

`chomp` を忘れると、入力した `1` が `"1\n"` となり、`$choice eq '1'` が false になります。

#### last でループ脱出
```perl
if ($choice eq '0') {
    last;  # while ループを抜ける
}
```

`while (1)` の無限ループから抜けるには `last` を使います。
